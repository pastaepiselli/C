# Esercizi C

Raccolta di esercizi C organizzati per argomento.

## Struttura

- `01_basi_e_variabili/`: primi programmi, tipi, costanti, variabili e conversioni.
- `02_operatori_e_calcoli/`: operatori, assegnamenti e calcoli semplici.
- `03_input_output_caratteri/`: input/output da carattere, conteggi e sostituzioni.
- `04_strutture_dati/`: array e strutture dati introduttive.
- `05_funzioni_e_scopo/`: funzioni, scope, esempi con funzioni di utilita.
- `06_esercizi_kr_e_avanzati/`: esercizi K&R e programmi piu avanzati.
- `build/`: eseguibili e output generati dalla compilazione.

## Compilazione

Compila un esercizio dalla root indicando il percorso del file:

```sh
gcc 05_funzioni_e_scopo/function.c -o build/function
```
